Para que se muestre la informacion de un link en facebook
Colocamos al inicio del archivo 'html' el 'prefix' 

        html lang="es" prefix="og: http://ogp.me/ns#"

Luego nos vamos al 'head' y colocamos lo siguiente:

        
          <head>
            <title>The Rock (1996)</title>
            <meta property="og:title" content="The Rock" />
            <meta property="og:type" content="video.movie" />
            <meta property="og:url" content="https://www.imdb.com/title/tt0117500/" />
            <meta property="og:image" content="https://ia.media-imdb.com/images/rock.jpg" />
            ...
          </head>

En cada etiqueta 'meta' agregamos un atributo 'property' con un valor como 'og:...', que define una propiedad y su tipo de dato.
En 'content=""' se coloca el valor correspondiente a la propiedad.
